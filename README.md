# Σύστημα Αξιολόγησης Προϊόντων

## 1. Περιγραφή:

Η παρούσα εργασία, υλοποιεί ένα απλό σύστημα αναζήτησης και αξιολόγησης προϊόντων σε περιβάλλον κονσόλας (console application) με χρήση της γλώσσας προγραμματισμού **Python**.

Ο απλός χρήστης μπορεί να:
- αναζητήσει προϊόντα με βάση το όνομα ή την κατηγορία τους,
- δει αναλυτικές πληροφορίες για κάθε προϊόν (ονομασία, κατηγορία, περιγραφή και διαθέσιμα ηλεκτρονικά καταστήματα με τις αντίστοιχες τιμές),
- προβάλει τις αξιολογήσεις ενός προϊόντος.

Οι εγγεγραμμένοι χρήστες μπορούν να συνδεθούν στο σύστημα και να καταχωρήσουν τη δική τους αξιολόγηση για ένα προϊόν, εισάγοντας:
- βαθμολογία από **1 έως 5 αστέρια**,
- ένα σχόλιο.

Κάθε χρήστης μπορεί να καταχωρήσει **μόνο μία αξιολόγηση ανά προϊόν**.

Όλα τα δεδομένα αποθηκεύονται προσωρινά στη μνήμη του προγράμματος και δεν χρησιμοποιείται βάση δεδομένων.
